# 📸 图片说明

## 如何添加项目展示图片

1. **替换 `aurora-main-repo.png`**：
   - 将您的原版Aurora仓库截图保存为此文件名
   - 建议尺寸：1200x800px
   - 格式：PNG或JPG

2. **替换 `aurora-file-structure.png`**：
   - 将您的项目文件结构截图保存为此文件名
   - 建议尺寸：1200x800px
   - 格式：PNG或JPG

## 图片要求

- **清晰度**：高分辨率，文字清晰可读
- **内容**：展示项目的完整性和专业性
- **格式**：PNG（推荐）或JPG
- **大小**：建议不超过2MB

## 注意事项

- 确保图片中没有敏感信息（如API密钥、密码等）
- 图片将公开展示，请确保内容适合公开
- 建议使用截图工具或专业设计软件制作
