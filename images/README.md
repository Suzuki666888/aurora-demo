# 📸 图片添加指南

## 🚨 重要提醒

**当前图片区域为占位符，需要您手动添加真实图片！**

## 📋 需要添加的图片

### 1. 原版仓库截图
- **文件名**: `aurora-main-repo.png`
- **内容**: GitHub仓库页面截图
- **尺寸**: 建议 1200x800px
- **格式**: PNG（推荐）或JPG

### 2. 项目文件结构截图
- **文件名**: `aurora-file-structure.png`  
- **内容**: 本地项目文件夹截图
- **尺寸**: 建议 1200x800px
- **格式**: PNG（推荐）或JPG

## 🛠️ 如何添加图片

1. **截图工具**：
   - macOS: `Cmd + Shift + 4` (区域截图)
   - Windows: `Win + Shift + S`
   - 或使用专业截图工具

2. **保存步骤**：
   - 截图后保存到 `images/` 目录
   - 使用指定的文件名
   - 确保格式正确

3. **检查要求**：
   - ✅ 图片清晰可读
   - ✅ 没有敏感信息
   - ✅ 文件大小 < 2MB
   - ✅ 格式为PNG/JPG

## ⚠️ 注意事项

- **隐私保护**: 确保截图中没有API密钥、密码等敏感信息
- **内容审查**: 图片将公开展示，确保内容适合公开
- **质量要求**: 高分辨率，文字清晰可读
